# Todo App — DeployEase

A clean, dark-themed React todo app with priority levels and progress tracking.

## Features
- Add tasks with High / Medium / Low priority
- Mark tasks complete with animated checkbox
- Filter by All / Active / Done
- Live progress bar and stats
- Delete tasks

## Tech Stack
- React 18
- Vite
- CSS (no UI library)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy
This project auto-deploys to GitHub Pages via GitHub Actions on every push to `main`.

Live URL: `https://<your-username>.github.io/todo-app/`
